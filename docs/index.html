What It Does

Veteran Core profiles

Ordinary gun turrets stay stackable until a Veteran Core is installed. The core carries XP, levels, upgrades, elements, and custom name.

Feeder input

Installing a core creates a nearby feeder inventory. Insert element resources into it manually or with inserters to complete material projects.

Combat feedback

Shows HP, shooting speed, range, loaded ammo, estimated ammo damage, estimated DPS, kills, damage dealt, and quality summaries for affected rows.

Early balance pass

Awards configurable XP from cored turret damage and proportional kill credit, then lets level-earned points buy compact core upgrades and augments.

Playtest Path

Install from the in-game Mods interface, place a vanilla gun turret, add ammo, open it, and confirm the Turret XP panel appears on the right.

  1. Install or dev-create a Veteran Core.
  2. Let the turret shoot enemies.
  3. Check that XP and lifetime damage increase.
  4. Name the core, toggle its floating label, and move it to another turret.
  5. Use dev buttons to test level gates, start and complete an element project, then buy an upgrade or augment.
  6. Report confusing stats, pacing issues, or runtime errors.
