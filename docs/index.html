What It Does

Veteran Cores

Only chosen turrets become unique. A core stores XP, upgrades, elements, name, and battle history, then moves by itself or bound to its turret body when the front line shifts.

Combat Growth

Turrets earn XP from damage and kill contribution. Points buy damage, repair, lifesteal, crits, range, luck, double shots, XP gain, and bouncing bullets.

Specialized Defenders

Choose strong identities with tradeoffs: long-range snipers, rapid machine guns, tough bulwarks, or brutal close-range brawlers.

Element Projects

Pick fire, electric, explosive, or toxic effects, feed larger passive material ranks to strengthen them, and combine two elements on veteran turrets.

Playtest Path

Install from the in-game Mods interface, place a gun turret, install a Veteran Core, add ammo, and let it fight.

  1. Install or dev-create a Veteran Core.
  2. Let the turret shoot enemies.
  3. Check that XP and lifetime damage increase.
  4. Name the core, toggle its floating label, and move it to another turret or bind it for one-item turret moves.
  5. Run /turret-xp-dev to test level gates, complete passive element material ranks, then buy an upgrade or augment.
  6. Report confusing stats, pacing issues, or runtime errors.
